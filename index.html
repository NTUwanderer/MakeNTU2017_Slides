Close Your Eyes

Why Do We Make It

Long Time Working With Eyes

  • Read books
  • Watch screens
  • Use cellphones
  • Near distance usage hurts eyes

What Does It Solve

On Your Glasses

Detect near distance usage on your glasses, record and analyze the usage on the cloud.

Real Time Show The Usage

Build a website to show users their eyes' usage.

Remind Users To Take Rest

When a user keep near distance usage for a while, the glasses may prevent the user from doing that and remind him/her to take rest.

How Do We Implement It

Arduino

Attach a sonar to detect the distance of the user's eyesight.

Attach two servomotors to prevent the user from keep using if necessary.

Asus Tinker Board

Communicate with Arduino, including handling the signals of the sonar and sending the on/off signals of motors.

Build a connection with MicroSoft Azure Cloud server with websocket to transmit the data and control signals.

MicroSoft Azure Cloud server

Data center, websocket's server and website's server. Store users' data, analyze the data, real time send updates to existing websites, and send the control signals of motors.

Website

Real time render the chart of eyes' usage to let users be aware of the current usage and further arrange the usage.
